# CareerGuide AI

CareerGuide AI is an end-to-end career development platform designed to help users prepare, assess, and improve their career readiness using AI-powered tools.  
It brings together resume building, skill assessment, and voice-based AI interviews into one unified experience.

---

## Features

### AI Resume Builder
- Create and manage multiple resumes
- Clean, structured resume content
- AI-assisted formatting and content generation

### Assessment Platform
- Skill-based quizzes (technical and behavioral)
- Instant scoring and feedback
- AI-generated improvement tips
- Track performance over time

### Voice AI Interview
- Simulated real-world interview experience
- Role-specific interview questions
- Voice-based user responses
- AI-generated interview feedback and evaluation
